Buession Framework HttpClient Changelog
===========================

Changes in version 2.0.0 (2022-02-22)
-------------------------------------
*【变化】重构底层代码，内部包结构做重大调整
*【移除】ObjectRequestBodyConverter
*【移除】okhttp 连接管理器 com.buession.httpclient.okhttp.OkHttpClientConnectionManager 类，使用 okhttp3.HttpClientConnectionManager，API okhttp 保持一致
*【移除】okhttp 连接管理器 com.buession.httpclient.okhttp.RequestBuilder 类，使用 okhttp3.RequestBuilder，API okhttp 保持一致

Changes in version 1.3.2 (2021-11-10)
-------------------------------------

Changes in version 1.3.1 (2021-07-27)
-------------------------------------
*【修复】ApacheClient ConnectionPoolTimeoutException 和 ConnectTimeoutException 判断顺序错误的问题
*【变化】com.buession.httpclient.okhttp.OkHttpClientConnectionManager 不推荐使用，使用 okhttp3.HttpClientConnectionManager 替代

Changes in version 1.3.0 (2021-07-11)
-------------------------------------
*【变化】移除静态方法上的 final 关键字
*【修复】HTTP 头构建器 HeadersBuilder 批量添加 HTTP 头无效的问题
*【变化】HttpClient API 实现带有 ConnectionManager 参数构造函数，调整为各自驱动 ConnectionManager
*【变化】重写核心 pojo 类 hashCode 和 equals 方法
*【管理】逐步规范化 changelog 和 测试用例
