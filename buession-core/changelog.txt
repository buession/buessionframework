Buession Framework Core Changelog
===========================


Changes in version 1.3.3 (2022-02-22)
-------------------------------------

Changes in version 1.3.2 (2021-11-10)
-------------------------------------
*【新增】Validate 新增 isMimeType 方法，验证是否为合法的 MimeType 格式

Changes in version 1.3.1 (2021-07-27)
-------------------------------------
*【新增】DateTime 新增获取当前 Unix Timestamp 方法
*【新增】List 构建器 ListBuilder、Map 构建器 MapBuilder、Set 构建器 SetBuilder、Queue 构建器 QueueBuilder
*【变化】移除静态方法上的 final 关键字
*【变化】MapUtils.toSet 当为 LinkedHashMap 实例时，返回 LinkedHashSet；其它 Map 实例时，返回 HashSet
*【新增】manager 层注解 @Manager
*【新增】基于 mapstruct 的 Pojo 映射接口
*【新增】ID 生成器

Changes in version 1.3.0 (2021-07-10)
-------------------------------------
*【变化】移除静态方法上的 final 关键字
*【新增】完善 ArrayUtils contains、indexOf、lastIndexOf、toString 方法对基础类型的操作
*【变化】ArrayUtils 继承 org.apache.commons.lang3.ArrayUtils
*【新增】Validate 新增 isBlank 验证字符串为 null、空字符串、空白字符
*【新增】Validate 新增 isNotBlank 验证字符串不为 null 且空不为字符串且不为空白字符
*【新增】Destroyable 接口
*【新增】Deprecated StatusBooleanConvert
*【管理】逐步规范化 changelog 和 测试用例
