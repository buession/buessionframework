Buession Framework Web Changelog
===========================


Changes in version 2.0.0 (2022-02-22)
-------------------------------------
*【移除】com.buession.web.reponse 包中的 response 注解，改由 com.buession.web.reponse.annotation 包中 response 注解替代
*【修复】AbstractController、AbstractRestController 支持 servlet 和 reactive 的兼容性问题
*【变化】注解 AOP 重构
*【修复】@Configuration 在 servlet 和 webflux 冲突的问题

-------------------------------------
